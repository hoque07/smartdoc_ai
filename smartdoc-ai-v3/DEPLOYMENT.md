# Deployment Guide

## Local Run

```bash
npm install
npm run dev
```

## GitHub Push

```bash
git init
git add .
git commit -m "Initial version"
git branch -M main
git remote add origin https://github.com/hoque07/smartdoc-ai.git
git push -u origin main
```

## Enable GitHub Pages

GitHub repository → Settings → Pages → Build and deployment → Source → GitHub Actions
